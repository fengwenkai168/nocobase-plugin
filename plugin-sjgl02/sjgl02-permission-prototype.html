📋 界面元素标注清单（红色数字①②③···对应页面中红色圆点）

  1. ① 左侧栏：用户/角色选择面板（span=6，260px）
  2. ② 搜索框：输入关键字实时过滤用户或角色名称
  3. ③ 角色分组：绿色头像 + 角色名，点击选中查看权限配置
  4. ④ 右侧权限配置区：选中目标的权限卡片区域（span=18）
  5. ⑤ 选中目标信息：头像 + 用户名 + 类型标签 + 所属角色标签
  6. ⑥ 全局设置：任务查看范围（仅看自己的 / 查看全部），立即保存
  7. ⑦ 添加权限按钮：弹出权限编辑弹窗，admin/root 角色时隐藏此按钮
  8. ⑧ 批量操作栏：全选自定义权限 + 批量删除（Popconfirm 确认）
  9. ⑨ 权限搜索框：按表名或表标识过滤权限卡片列表
  10. ⑩ 系统管理提示：admin/root 角色专属 Alert 提示，全部权限不可修改
  11. ⑪ 继承权限分区：从角色继承来的权限，灰色背景、不可编辑，仅可查看详情
  12. ⑫ 权限标记：紫色="继承"、蓝色="系统管理"
  13. ⑬ 权限标签行：蓝色(导入) / 绿色(导出) / 橙色(模式+唯一值) / 红色(必填) / 青色(可导入数) / 紫色(可导出数) / 灰色(筛选条件)
  14. ⑭ 自定义权限分区：当前目标的独立权限配置，可编辑、可删除
  15. ⑮ 批量选择框：勾选后通过"批量删除"按钮操作，仅自定义权限可选
  16. ⑯ 编辑/删除按钮：单条权限的操作入口（编辑→弹窗，删除→即时移除）
  17. ⑱ 数量化标签 ★ 缺失：卡片上显示可导入/可导出字段数量
  18. ⑲ 分页控件：每页 10 条，总条数超过时自动分页
  19. ⑳ 权限编辑弹窗：Modal 720px，包含所有权限配置项
  20. ㉑ 数据表选择器：搜索过滤 + "标题（表名）"格式，选中后自动加载字段
  21. ㉒ 允许导入/导出开关：分别控制下方导入/导出配置区域显示/隐藏
  22. ㉓ 导入配置区：蓝色左边框标识，Switch 开→显示，关→隐藏
  23. ㉔ 导入模式（多选）：支持同时勾选多种模式（insert/update/upsert），Tag 交互
  24. ㉕ 唯一值字段 ★ 实现偏离：应为 Tag 交互（已选可删 + 下拉标灰 + 实时计数），当前为普通多选
  25. ㉖ 导出配置区：绿色左边框标识，Switch 开→显示，关→隐藏
  26. ㉗ 导出范围筛选 ★★★ 完全缺失：多行 AND 条件（字段→操作符→值输入），支持添加/删除行
  27. ㉘ 查看详情弹窗（新增）：继承权限只读查看，显示完整配置表 + 来源角色提示 + 所有字段详情
  28. ㉙ 子Tab切换：权限配置（编辑模式）/ 操作日志（审计模式）
  29. ㉚ 日志筛选栏：按操作类型（创建/修改/删除/切换）、时间范围筛选审计日志
  30. ㉛ 操作日志表：7 列完整审计（时间/操作人/操作/目标/数据表/变更概要/详情展开），支持展开查看完整变更 JSON
